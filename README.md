## Overview

Godesk is a Windows optimization app focused on control, transparency, and everyday usability. It brings together cleanup tools, privacy-conscious tweaks, performance settings, system restore workflows, hardware metrics, and update/licensing support in a single desktop experience.

The goal is simple: help users keep their PC clean, responsive, and understandable without relying on heavy background services, hidden telemetry, or risky one-way changes.

## Features

| Area | What Godesk provides |
| --- | --- |
| **Dashboard** | Quick system overview with CPU, memory, disk, and hardware information. |
| **Hardware monitoring** | Real-time metrics powered by a native bridge and LibreHardwareMonitor. |
| **Debloat tools** | Remove or manage unnecessary Windows apps and bundled components. |
| **Performance tweaks** | Apply general, network, graphics, memory, timer, and gaming-oriented optimizations. |
| **System restore** | Create and manage restore points before applying sensitive changes. |
| **Disk cleanup** | Launch cleanup flows to reclaim storage and reduce clutter. |
| **Fast transfer** | File transfer workflow for moving files quickly from the desktop app. |
| **Benchmarking** | Lightweight benchmark utilities for checking system behavior. |
| **Updates** | GitHub Releases based update flow with downloaded installer preparation. |
| **Plus licensing** | Google sign-in and paid-license validation through the official web backend. |

## Product Principles

- **Transparent by design:** changes are visible and intentionally scoped.
- **Safety first:** restore workflows help users roll back when needed.
- **Low overhead:** the app is designed to stay lightweight when minimized.
- **Built for Windows:** native helpers handle privileged actions and hardware access.
- **No hidden telemetry:** the app does not collect or send user data.
